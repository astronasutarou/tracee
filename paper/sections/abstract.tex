\begin{abstract}
  An efficient algorithm is required to extract moving objects (asteroids, satellites, and space debris) from enormous data with advances in observational instruments. We have developed an algorithm, \texttt{tracee}, to swiftly detect points aligned as a line segment from a three-dimensional space. The algorithm consists of two steps; First, construct a k-nearest neighbor graph of given points, and then extract colinear line segments by grouping. The proposed algorithm is robust against distractors and works properly even when line segments are crossed. The fast-moving object processing pipeline of the Tomo-e Gozen project, which utilizes the proposed algorithm, has detected about 30 near-earth asteroids by August 2020. The algorithm can be used in other uses, for example, to trace ejecta by impact experiments.
\end{abstract}
\keyword{Software, Algorithms, Asteroids, Satellites}
